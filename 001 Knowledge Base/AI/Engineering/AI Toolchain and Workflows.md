## AI Toolchain and Workflows

*Living document tracking complete AI-powered workflow*

### Core AI Interface
- **Warp**: AI-enhanced terminal and coding environment for development and non-coding tasks
  - Agent Mode for interaction
  - Voice instructions for efficient prompts
- **Claude Client**: Conversational AI and MCP client 
  - Pro subscription

### Browser & Content Processing
- **Max AI**: Website summarization and translation
- **NotebookLM**: Content analysis for books and videos

### Memory & Knowledge Management
- **Pieces**: Procedural memory for code snippets
- **Obsidian + MCP**: AI memory vault for knowledge storage

### Key MCP Servers
- **Obsidian MCP**: AI memory integration
- **Linear MCP**: Project management
- **Tavily MCP**: Web search
- **Pieces MCP**: Procedural memory interaction
- **HuggingFace MCP**: ML model and dataset discovery
- **Context7 MCP**: Technical documentation
- **Playwright MCP**: Browser automation and testing

### Archive
- **Cursor**: AI IDE as VSCode clone (not in use any more)

### Honorable mentions
- **neo:** A data-focused VSCode clone like Cursor

## Common Workflow Scenarios

### Coding Workflow
**Context**: Developing new features
1. **Warp Agent Mode**: Describe requirements
2. **Tavily MCP**: Web search
3. **Linear MCP**: Update tasks
4. **Context7 MCP**: Retrieve docs
5. **Pieces**: Access code snippets
6. **Obsidian MCP**: Store learnings

### Documentation Workflow
**Context**: Document after meetings
1. Dictate notes to Warp
2. Store artifact in Obsidian
3. Warp enriches documentation
4. Finalize and store
5. (Optional) Commit to git

### Task Management Workflow
**Context**: Organizing tasks
1. **Linear MCP**: Fetch current tasks
2. **Claude Client**: Analyze priorities
3. **Warp**: Update tasks
4. **Obsidian MCP**: Store strategic decisions

### Research and Learning Workflow
**Context**: Conducting research
1. **Warp**: Initiate search with voice
2. **Tavily MCP**: Perform deep web search
3. **NotebookLM**: Analyze content
4. **Obsidian MCP**: Record insights
5. **Pieces**: Save valuable snippets
