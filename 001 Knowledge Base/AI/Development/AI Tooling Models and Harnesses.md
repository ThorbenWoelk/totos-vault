It's not only about the models to, e.g., vibe code but also about which harness/client you use to run them - Claude Code, Cursor, Antigravity, Warp, Kilo, Crush, Windsurf, Mistral vibe CLI, gemini CLI ... 

These don't necessarily get better all the time. Sometimes, like in early 2025, they actually decline in quality (e.g. Claude Code), before a new model comes out and they cook again (end 2025 ralph wiggum pattern and Claude Code taking over). 

Frequently changing providers and harnesses makes it all the more challenging for companies that usually have a long security and approval process to allow the use of a new technology.
