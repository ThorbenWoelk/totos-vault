# Linear Archive

This folder stores archived backups of completed Linear issues exported from the workspace by a local background sync service.

- `Issues/YYYY/YYYY-MM/` contains one Markdown note and one JSON snapshot per issue.
- The files are generated automatically and updated when a completed issue changes in Linear.
